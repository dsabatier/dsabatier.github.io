Art & Experiments

A collection of standalone UI experiments.

Diagetic Dialog

A complete dialog system with full-featured, gamepad ready, diagetic widgets supporting branching dialogs.

Fake 3D Effect

An experiment creating a faux-3D effect in Unreal 5 UI. This works by using a custom HLSL node to warp the vertices of the quad and then applying a perspective transform to the texture. It's a neat trick to get a 3D effect without using a 3D model.

LTTP Remake UI

Menu system and heads-up display recreating the classic A Link to the Past aesthetic. Despite the low-res look, this is a fully functional menu system using Common UI + Enhanced Input to provide keyboard and controller support (including updating UI hints based on input type).

Animated List View

An animated list view using Common UI and MVVM. Each 'option' has a VM, and the list has a VM holding those.

Halftone Material

A halftone material implementation for the UI. Can be animated, rotated, scaled, and more.

The effect is extracted into its own function, making it easy to apply to any UI material.

Any SDF can be passed in to create custom patterns (Square in this case).

Being computed on the GPU, we can animate it in interesting ways with very little cost.

Figma Inventory Flow Prototype

A Figma prototype for an inventory system, fully navigable with mouse, keyboard, and gamepad.

Try it out with a gamepad! (Try gamepad face buttons for inventory actions)
